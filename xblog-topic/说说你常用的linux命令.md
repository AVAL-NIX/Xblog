讲解一个运维的整个生命周期吧.

- rz  上传
- sz  下载
- tar -zxvf 解压
- tar -zcvf 压缩
- cp 复制
- mv 移动
- ps aux | grep java 查看java端口
- whereis / find / which 找文件位置
- vi vim 编辑文件 :wq! 保存文件
- kill -9 pid 杀掉进程
- top 看CPU
- netstat -anp | greap 'xxx' 看网络端口
- username 看名字 .
